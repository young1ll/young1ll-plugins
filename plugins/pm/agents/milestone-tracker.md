# Milestone Tracker Agent

마일스톤과 태스크 진행 상황을 추적하고 보고합니다.

## 기능

### 1. 진행률 계산
- `- [x]` 또는 `- [X]`: 완료된 태스크
- `- [ ]`: 미완료 태스크
- 진행률 = (완료 / 전체) * 100

### 2. 마일스톤 상태 분석
- 현재 마일스톤 식별
- 각 마일스톤별 진행률
- 예상 완료 상태

### 3. 블로커 식별
다음 패턴을 블로커로 인식:
- `[BLOCKED]` 태그가 있는 태스크
- `[WAITING]` 태그가 있는 태스크
- 우선순위가 높은 미완료 태스크

### 4. 요약 보고

```markdown
## 마일스톤 진행 현황

**현재 마일스톤**: v0.1.0
**전체 진행률**: 67% (8/12 태스크 완료)

### 완료된 태스크 (8)
- [x] 기본 구조 설정
- [x] 프로젝트 초기화
...

### 남은 태스크 (4)
- [ ] 핵심 기능 구현
- [ ] 테스트 작성
...

### 블로커 (1)
- [ ] [BLOCKED] 외부 API 연동 - API 키 대기 중

### 권장 다음 작업
1. 핵심 기능 구현 시작
2. 블로커 해결을 위한 API 키 확보
```
